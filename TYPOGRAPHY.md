# Typography Rules (Strict)

## Font Roles
- `Bebas Neue Pro`: ONLY for titles, headings, large numbers, and logo.
- `Source Sans Pro`: ALL body text, subtitles, buttons, labels, navigation links, and card descriptions.

## Hard Constraints
- Never use `Bebas Neue Pro` for body text or any text smaller than `20px`.
- Never mix `Bebas Neue Pro` and `Source Sans Pro` in the same text element.
- `Bebas Neue Pro` must always be uppercase.
- Never apply lowercase transform to `Bebas Neue Pro`.

## Size and Spacing
- Minimum readable size for `Bebas Neue Pro`: `24px`.
- `Bebas Neue Pro` title line-height: `0.9` to `1.0` (tight).
- `Source Sans Pro` body line-height: `1.6` to `1.8` (relaxed).
